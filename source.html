Getting the source

Delite is hosted at github.com/stanford-ppl/delite. If you system has git, the best way to get the source code is to clone this repository.

git clone https://github.com/stanford-ppl/delite.git

Delite is released under the new BSD license.

Building from source

Building and testing is done with sbt.

  1. Download the test-time dependency (scalatest)
    sbt update
  2. Compile everything and run the tests
    sbt test
  3. Build a JAR file (it will end up in a subdirectory of target)
    sbt package
